## Build Setup

``` bash
# Setup local environment
$ python -m venv myenv

# Activated local environment
# For windows
$ myenv\Scripts\activate

# For mac os or linux
$ source myenv/bin/activate

# install dependencies
$ pip install -r requirements.txt

# update pip package
$ pip freeze > requirements.txt

# how to run
$ python main.py

# or run scripts
$ . bin/run.sh
```
